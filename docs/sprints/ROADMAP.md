# EVCharger Sprint Roadmap

**Project Start:** 2026-01-03  
**Target MVP:** Sprint 3 complete  
**Target App Store:** Post-Sprint 4

---

## Sprint Timeline

| Sprint | Duration | Goal | Key Deliverables |
|--------|----------|------|------------------|
| **Sprint 1** | ~5 days | Foundation | Xcode project, models, API service, CarPlay scaffold |
| **Sprint 2** | ~5 days | Core Features | CarPlay UI, nearby list, navigation handoff |
| **Sprint 3** | ~4 days | Polish & Premium | Settings UI, IAP, caching, offline mode |
| **Sprint 4** | ~3 days | Release Prep | App Store assets, entitlement, TestFlight |

---

## Sprint 1: Foundation

**Goal:** Establish project structure and core data layer

**Success Criteria:**
- [ ] Xcode project builds and runs
- [ ] Data models compile
- [ ] Open Charge Map API returns data
- [ ] CarPlay entitlement requested

---

## Sprint 2: Core Features

**Goal:** Working CarPlay experience end-to-end

**Success Criteria:**
- [ ] CarPlay shows nearby chargers
- [ ] Tap charger → see details
- [ ] Tap navigate → opens Apple Maps
- [ ] Connector filter works

## Sprint 3: Polish & Premium

**Goal:** Production-ready app with monetization

**Success Criteria:**

- [ ] Premium IAP flows work
- [ ] 24-hour caching implemented
- [ ] Works offline with cached data
